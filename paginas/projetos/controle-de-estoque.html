Controle de Estoque

C#, .NET, SQL Server

Programa para controle de estoque para uma loja de roupas. Integração do programa .NET com banco de dados SQL Server e programação em C#. Projeto feito em 2023 para o TCC do meu técnico em Desenvolvimento de Sistemas. Possui ferramentas para pesquisar, adicionar e alterar pedidos e clientes. Além de um guia de usuário para cada tela.
